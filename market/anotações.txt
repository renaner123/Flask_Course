Windows:
 Setar ambiente -> $env:FLASK_APP = "market"
 Ativar debug -> $env:FLASK_DEBUG = 1  
 Criar flask app-> set FLASK_APP=market.py  
 Executar flask app -> python -m flask run 


Instalar SQL from python:
    pip install flask-sqlalchemy
